# Задание 1

## Задача 1

### Текст задачи

Разработать класс с тремя целочисленными полями. Создать конструктор копирования. Разработать метод, вычисляющий минимальное из полей. Перегрузить метод ToString() для формирования строки из полей класса. Реализовать дочерний класс (его содержание предложить самостоятельно и описать в решении: какой содержательный смысл имеют поля; реализовать конструкторы; предложить и реализовать 2-3 метода). Протестировать все конструкторы и другие методы базового и дочернего классов. 

### Алгоритм решения

Программа создаёт два треугольника: первый имеет заданные стороны (6, 8, 10), а стороны второго задаются пользователем. После этого к обоим треугольникам применяются метод вывода и метод поиска минимальной стороны треугольника. Создаются два объекта дочернего класса треугольника (класса Tribool). С помощью методов этого класса треугольники анализируются на то, являются ли они вообще трегольниками и являются ли они трегольниками Пифагора. Если объект является треугольником, находиться его периметр.
Класс Triangle состоит из: трёх полей типа int, конструктора с параметрами и конструктора копирования, свойств для чтения и перезаписи полей, метода GetMinSide для поиска минимаьного по значению поля и перегруженного метода ToString. 
Дочерний класс Tribool состоит из полей, свойств и методов родительского красса, а так же собственных двух полей класса bool, одного конструктора, свойств для чтения полей, метода поиска периметра и метода, который выводит трактовку значения личных полей класса.

## Задача 2

### Текст задачи

Класс: RightTriangle. Метод: 
Унарные операции: 
++ увеличивает стороны треугольника в 2 раза (чтобы вновь получить катеты прямоугольного треугольника); -- уменьшает стороны треугольника в 2 раза (чтобы вновь получить 
катеты прямоугольного треугольника); 
Операции приведения типа: 
double (явная) – результатом является площадь треугольника, если треугольник существует и отрицательное число в противном случае; 
bool (неявная) – результатом является true, если треугольник с такими длинами сторон существует и falseв противном случае. 
Бинарные операции: 
<= Trianglet1, Trianglet2 – сравнивает площади треугольников; 
>= Trianglet1, Trianglet2 - сравнивает площади треугольников. 


### Алгоритм решения

Программа создаёт по данным пользователя два объекта класса RightTriangle. Один объект состоит из двух катетов прямоугольного треугольника. Оба треугольника выводятся на экран. К первому применяется унарная операция "--", ко второму унарная операция "++". Оба треугольника выводятся на экран. С помощью операции double находится площадь треугольников и выводиться на экран, с помощью бинарных операций "<=" и ">=" находится треугольник с большей площадью.
